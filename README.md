# Udacity-Facebook-final-lab
Classification of Flowers

## Prerequisites
- Python3.5+
- PyTorch 0.4.0

There are 102 classes of Flowers. The dataset has been uploaded. Here, I have used pretrained Resnet-101 for clasifying.
